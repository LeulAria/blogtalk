# Getting Started
### Clone The Project
```

$ git clone https://github.com/LeulAria/blogtalk
$ cd blogtalk

```

### Install Requirements/Project Dependencys
```

$ pip3 install -r requirements.txt
```

### Migrate Models
```

$ python3 manage.py migrate

```

## Run Server
```

$ python3 manage.py runserver

```
